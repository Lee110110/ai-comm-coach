# AI 沟通教练

> 在你说错话之前，帮你找到对的话。

一款 AI 驱动的沟通教练产品，帮助用户处理困难对话场景。无论是拒绝加班、谈涨薪、还是和伴侣沟通，AI 沟通教练都能即时给出专业建议和实战练习。

## 功能概览

### 🎯 急救话术

遇到困难对话，不知道怎么开口？描述场景，AI 立刻给出 3 种应对策略 + 雷区提醒 + 对方反应预判。

- 输入场景描述 → 获取委婉/直接/策略性三种方案
- 每个方案附带可直接使用的话术
- 雷区提醒帮你避开常见错误
- 预测对方可能的反应及应对方式

### 🎭 模拟演练

和 AI 角色扮演，反复练习困难对话，不怕说错话。

- AI 扮演你的领导/伴侣/客户，真实感对话
- 每句话实时反馈：做得好的 + 改进建议
- 三档难度：温和/中等/困难
- 结束后生成复盘报告，含维度评分和练习建议

### ✏️ 消息润色

写了一条消息，觉得语气不对？AI 帮你优化措辞。

- 输入原始消息 → 获取润色版本
- 逐条标注修改原因（为什么改、改了有什么效果）
- 提供不同语气的替代版本（正式/亲切/坚定/委婉）

### 📊 沟通画像

长期追踪你的沟通模式，发现自己的盲区。

- 6 维度评估：坚定程度、共情能力、表达清晰度、灵活变通、冲突处理、积极倾听
- 自动识别沟通盲区（如冲突回避、过度妥协）
- 趋势追踪，看到自己的成长轨迹

### 👥 关系档案

为每个重要的人维护专属沟通策略。

- 记录对方的沟通偏好、雷区、注意事项
- AI 基于档案生成专属沟通策略
- 互动记录时间线，回溯历史

## 快速开始

### 环境要求

- Python 3.10+
- Node.js 18+
- 一个 AI 大模型的 API Key（推荐 DeepSeek，性价比高）

### 1. 获取项目

```bash
cd C:\Users\Elvis\ai-comm-coach
```

### 2. 配置后端

```bash
cd backend

# 安装依赖
pip install -r requirements.txt

# 编辑配置文件
# 用你喜欢的编辑器打开 .env 文件
```

**必须修改的配置**（`.env` 文件）：

```ini
# 填入你的 AI 大模型 API Key
LLM_API_KEY=sk-xxxxxxxxxxxxxxxx

# 根据你使用的模型服务修改以下配置
# DeepSeek（推荐，性价比高）
LLM_BASE_URL=https://api.deepseek.com/v1
LLM_MODEL=deepseek-chat

# 智谱 AI
# LLM_BASE_URL=https://open.bigmodel.cn/api/paas/v4
# LLM_MODEL=glm-4-flash

# OpenAI
# LLM_BASE_URL=https://api.openai.com/v1
# LLM_MODEL=gpt-4o-mini
```

> 💡 **如何获取 API Key？**
> - DeepSeek：访问 [platform.deepseek.com](https://platform.deepseek.com)，注册后在 API Keys 页面创建
> - 智谱 AI：访问 [open.bigmodel.cn](https://open.bigmodel.cn)，注册后在 API Keys 页面创建
> - OpenAI：访问 [platform.openai.com](https://platform.openai.com)，注册后在 API Keys 页面创建

### 3. 配置前端

```bash
cd frontend

# 安装依赖
npm install
```

### 4. 启动项目

**开两个终端窗口：**

终端 1 — 启动后端：
```bash
cd ~\ai-comm-coach\backend
PYTHONPATH=. python -m uvicorn app.main:app --port 8000 --reload
```

终端 2 — 启动前端：
```bash
cd ~\ai-comm-coach\frontend
npm run dev
```

### 5. 访问应用

打开浏览器访问 **http://localhost:3000**

首次使用需要注册账号，注册后自动登录进入主页。

## 使用指南

### 第一次使用

1. **注册账号** — 打开应用后点击「注册」，填写用户名和密码
2. **进入首页** — 注册成功后自动跳转到 Dashboard

### 场景一：遇到困难对话，需要即时建议

1. 点击左侧菜单「急救话术」
2. 填写场景信息：
   - **场景标题**：如"拒绝领导加班要求"
   - **场景描述**：详细描述你的情况，越具体建议越精准
   - **补充背景**：如"已经连续加班两周了"（可选）
   - **关系类型**：选择对方是谁（上司/同事/伴侣/家人/朋友/客户）
   - **紧急程度**：低/中/高
3. 点击「获取建议」
4. 等待 3-5 秒，AI 会返回：
   - 🟢 **委婉策略** — 先共情再设边界
   - 🔵 **直接策略** — 明确拒绝 + 替代方案
   - 🟠 **策略性方案** — 把问题抛回给对方
   - ⚠️ **雷区提醒** — 这些话千万别说
   - 🔮 **反应预判** — 对方可能怎么回，你怎么接
5. 选择最适合你的方案，复制话术直接使用

### 场景二：想练习困难对话

1. 点击左侧菜单「模拟演练」
2. 点击「开始新演练」
3. 填写练习设置：
   - **场景描述**：如"和领导谈涨薪"
   - **角色描述**：如"你的领导，性格强势，注重结果"（越具体越好）
   - **难度**：建议从「温和」开始
4. 点击「开始」，AI 会先开口说话
5. 你回复后，AI 会：
   - 以对方角色回应你（真实感，不会配合你）
   - 在右侧反馈面板给你实时评价
6. 觉得难度不够？随时切换难度
7. 练习够了，点击「结束演练」
8. 查看复盘报告：总体评分、优点、待改进、各维度得分

### 场景三：消息措辞拿不准

1. 点击左侧菜单「消息润色」
2. 输入你写的原始消息
3. 可选：填写上下文、选择期望语气
4. 点击「润色消息」
5. AI 返回：
   - 润色后的版本（可一键复制）
   - 逐条修改说明（为什么改、改了有什么效果）
   - 不同语气的替代版本

### 场景四：了解自己的沟通模式

1. 点击左侧菜单「沟通画像」
2. 首次使用需要先有一些互动数据（急救话术/模拟演练/消息润色记录）
3. 点击「重新计算」，AI 会分析你的历史互动
4. 查看结果：
   - 🕸️ **雷达图**：6 个维度的整体画像
   - 📈 **趋势图**：各维度随时间的变化
   - 🔴 **盲区**：你反复出现的沟通问题
   - 🟢 **优势**：你擅长的沟通方式
   - 🔵 **趋势**：你最近的变化方向
5. 根据建议针对性练习

### 场景五：为某人维护专属沟通策略

1. 点击左侧菜单「关系档案」
2. 点击「添加关系」
3. 填写对方信息：
   - 姓名、关系类型
   - 沟通风格（如"喜欢数据支撑"、"不喜欢寒暄"）
   - 偏好（如"要点式沟通"、"先给结论"）
   - 避免话题（如"不要提XX"）
4. 保存后，点击进入详情页
5. 点击「获取沟通策略」— AI 会基于档案生成专属建议
6. 在急救话术/消息润色中选择关联此人，AI 会参考档案给出更精准的建议

## 常见问题

### Q: 启动后端报错 `ModuleNotFoundError`

确保使用 `PYTHONPATH=.` 参数启动：

```bash
cd backend
PYTHONPATH=. python -m uvicorn app.main:app --port 8000 --reload
```

### Q: AI 建议生成很慢

正常现象，AI 生成需要 3-10 秒。如果超过 30 秒，检查：
- API Key 是否正确
- 网络是否能访问 AI 服务
- `.env` 中的 `LLM_BASE_URL` 是否正确

### Q: AI 返回的内容质量不好

- **场景描述越具体，建议越精准**。不要只写"和领导沟通"，要写"领导让我周末加班做PPT，但我已经有家庭安排，而且这周已经加了3天班"
- 推荐使用 DeepSeek 或 GPT-4 级别的模型，小模型效果会打折扣

### Q: 如何切换 AI 模型？

编辑 `backend/.env` 文件，修改 `LLM_BASE_URL`、`LLM_API_KEY`、`LLM_MODEL` 三个配置，重启后端即可。

### Q: 数据存在哪里？

所有数据存储在 `backend/data/app.db`（SQLite 文件）。删除此文件会清空所有数据。

### Q: 前端页面空白

确保后端已启动且运行在 8000 端口，前端的开发代理会将 `/api` 请求转发到后端。
